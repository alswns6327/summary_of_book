# 5장 타입 활용하기

제네릭을 사용해서 extends 키워드를 한정자로도 사용가능하다.

```typescript
type A<T> = T extends B ? B : C; // 제네릭 T가 B를 확장한 형태면 B, 아니면 C를 타입으로 지정
let a: A<B>; // a는 B 타입
let a2: A<D>; // a2는 C 타입

type A2<T extends B> = T; // 제네릭 T는 B를 확장한 형태만 올 수 있다.
```
